Le fichier principal est serveur.py

dns_db_create.py permet de créer une base de données avec SQLite pour y mettre les IP des
noms de domaine

access_db.py est un fichier 'tutoriel' pour montrer comment accéder aux données dans une db SQLite

dns_db.db est la base de données SQLite

Demo.avi est une courte démonstration du serveur DNS

ExServeur.pcap est le fichier pcap des paquets DNS récupérés lors de la démonstration

INSTALLATION:

-Lancer la capture en compilant le server.py 
-Laisser le server tourner pour capturer les paquets
-Executer des requetes DNS et DHCP 
-Consulter le fichier log du jour et la blacklist pour les logs non autorisés
-Lancer le clientDatabase.py puis taper le numéro de l'action souhaitée pour afficher les logs
